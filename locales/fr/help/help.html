Menu simple

Le plugin Simple Menu est utilisé par le thème pour afficher le menu du haut de page.

Astuce : pour ajouter un lien vers le bloc des widgets, vous pouvez créer un élément de menu libre avec un lien vers #menubar

Image de fond

Vous pouvez choisir d'afficher l'image par défaut pour toutes les pages ou afficher une image aléatoire à chaque chargement de page.

Si le plugin Featured Media est installé, vous pouvez choisir d'utiliser l'image à la une pour vos billets.

Choix des images

Vous pouvez changer chacune des images. Cliquez pour cela sur le bouton "Changer", choisissez l'image dans votre médiathèque et enregistrez. Cliquez sur "Réinitialiser" pour utiliser à nouveau l'image fournie avec le thème.

Les dimensions des images doivent être au moins de 1200*450px. Seule la taille originale peut être sélectionnée.

Liens vers les médias sociaux

Les étiquettes s'afficheront en bas des pages. Vous devez indiquer pour chacune un libellé et l'URL correspondante. (Laissez l'un de ces deux champs vide pour ne pas afficher l'étiquette).

Vous pouvez également modifier l'ordre d'affichage des étiquettes par tiré-lâché si javascript est actif et que vous ne l'avez pas interdit dans vos préférences. Si javascript est désactivé, un champ supplémentaire permet d'indiquer l'ordre d'affichage (de 1 à n).
