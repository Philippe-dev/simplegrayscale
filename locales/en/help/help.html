Simple Menu

Simple Menu plugin is used by the theme to manage the top menu.

Tip : to add a link to the widgets block, create a free menu item with a link to #menubar

Background image

You can choose to display the default image on all pages, or a random image at each page load

Choice of images

You can replace any image. Click on the "Change" button and choose an image in your media manager, then save your changes. Click on the "Reset" button to use again any of the theme's default images.
